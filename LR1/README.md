# Задача

Реализация базовых структур данных на C++ и Rust с CLI и тестами:

- a. `Array` — динамический массив  
- b. `LinkedList` — односвязный список  
- c. `DoublyLinkedList` — двусвязный список  
- d. `Stack` — стек  
- e. `Queue` — очередь  
- f. `Red-Black Tree` — красно-черное дерево
